# homebrew-boso

Homebrew tap for [Boso AI](https://boso.com) — self-hosted coding agent.

## Install

```sh
brew tap e01061978-sketch/boso
brew install boso
```

## Run

```sh
boso
```

Defaults to the public Boso API at `https://ai.boso.com/v1`. Requires an API key; see [the main repo](https://github.com/e01061978-sketch/boso-ai).
